# Javascript简明教程
   
这是一份大纲式的Javascript培训教材， 建议有一定js基础之后再进行阅读。
   
#### JS是一种脚本语言
   
JS是运行在客户端的脚本，用户的浏览器就是他的执行平台，而不需要依靠其他任何设备或者软件，浏览器称为javascript的运行环境，也叫做宿主环境。

浏览器只是js的其中一种宿主环境，比如flash同样支持js代码，所以flash player也是js的一种宿主环境
